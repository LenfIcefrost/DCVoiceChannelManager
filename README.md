# DCVoiceChannelManager

Help member create voice channel and manage channel themself

### Requirement
python3.8 later

## Installation

```shell
pip install discord.py
```

## Setting before use
1. find static_data.py 
2. set bot_token in Line 5



## How to use
```shell
python main.py
```
